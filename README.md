## BOOLFIX
Tecnologie
- Vue Js
- HTML5
- CSS3

### Caratterisiche
- Ricerca film tramite l'input in alto a destra
- Visualizzazione informazioni e breve trama del film al passaggio del mouse sulla scheda del film
